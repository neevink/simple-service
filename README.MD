# Бэкенд для социальной сети

### Локальный запуск
Установить переменную окружения, зависимости, активировать переменную окружения:
```shell
make devenv
```
Запустить сервер:
```shell
app
```

### Тесты и линтер
Запуск тестов:
```shell
make test
```
Запуск линтера для проверки качества кода:
```shell
make lint
```
